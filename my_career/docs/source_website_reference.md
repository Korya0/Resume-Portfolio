# Source Website Reference

This project is a Flutter conversion of the "MY CAREER" HTML/CSS/JS website.
The full source HTML was provided in the initial conversation and is documented
in the implementation plan artifact.

## Key Design Specs
- Dark theme only: bg=#08090f, surface=#0f1118
- Fonts: Space Mono (mono), DM Sans (body)
- Accent: #00d4ff (cyan), #7c3aed (purple)
- 6 pages: Roadmap, Books, Tips, Problems, Notes, CV Compare
- Sidebar with external links
- All data persisted via localStorage (→ SharedPreferences)
